# Online Complaint Management System with Location Intelligence

## 📌 Overview

The Online Complaint Management System is a web-based platform designed to register, track, and resolve citizen complaints with integrated real-time location services. The system captures geographic coordinates using location APIs and organizes complaints through a structured workflow to improve monitoring and response efficiency.

This project demonstrates practical skills in **backend development, API integration, database design, and scalable system architecture**.

---

## 🚀 Features

* Online complaint registration and status tracking
* Real-time geolocation capture using location APIs
* Role-based access for administrators and users
* Centralized dashboard for complaint monitoring
* Structured workflow for complaint resolution
* Secure data storage and authentication

## ⚙️ System Architecture

The system follows a layered architecture:

1. **User Interface Layer** – Complaint submission and tracking portal
2. **API Integration Layer** – Location services for capturing coordinates
3. **Application Layer** – Django backend managing business logic
4. **Database Layer** – PostgreSQL storing complaint and user records

## ▶️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/DIVAKAR-0110/SOA.git

# Navigate to project directory
cd SOA

# Install dependencies
pip install -r requirements.txt

# Run the server
python manage.py runserver
```
